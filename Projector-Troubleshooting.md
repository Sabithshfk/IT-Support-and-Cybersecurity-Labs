# Projector Troubleshooting Guide

## Problem
Projector shows no display.

## Troubleshooting Steps

### 1. Check Power
- Verify projector is powered on.
- Check power cable and power indicator lights.

### 2. Check Connections
- Verify HDMI/VGA cable is securely connected.
- Inspect cable for damage.
- Try a known-good cable.

### 3. Verify Input Source
- Check if projector is set to the correct input source.
- Example: HDMI 1, HDMI 2, VGA.

### 4. Check Laptop Display Settings
- Press Win + P.
- Select Duplicate or Extend.
- Verify external display is detected.

### 5. Test Resolution
- Lower display resolution.
- Ensure projector supports the selected resolution.

### 6. Test with Another Device
- Connect another laptop.
- Determine whether issue is with projector or laptop.

### 7. Check Projector Hardware
- Verify lamp status.
- Check for warning indicators.
- Restart projector.

## Interview Answer

If a projector shows no display, I would first verify power and cable connections. Then I would check the projector input source and laptop display settings. If the issue remains, I would test with a known-good cable and another laptop to isolate whether the problem is caused by the projector, cable, or laptop.
