# Active Directory Basics

## What is Active Directory?

Active Directory (AD) is a Microsoft directory service used to manage users, computers, groups, and resources within an organization.

## Common Tasks

### User Management
- Create users
- Disable users
- Reset passwords
- Unlock accounts

### Computer Management
- Join computers to a domain
- Remove computers from a domain

### Group Management
- Create security groups
- Assign permissions

## Benefits

- Centralized management
- Improved security
- Easier administration

## Common Interview Question

### What is Active Directory?

Active Directory is a Microsoft directory service that provides centralized management of users, computers, groups, and resources within a network environment.

### What are common IT Support tasks in Active Directory?

Common tasks include password resets, unlocking accounts, creating users, disabling accounts, and managing group memberships.
